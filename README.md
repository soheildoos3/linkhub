## ✨ قابلیت‌های کلیدی

| قابلیت                 | توضیح                                                    |
| ---------------------- | -------------------------------------------------------- |
| 📄 **صفحه عمومی شخصی** | هر کاربر یک صفحه اختصاصی برای لینک‌های خود دارد          |
| 🎯 **مدیریت لینک‌ها**  | افزودن، ویرایش، حذف و مرتب‌سازی با drag & drop           |
| 🌐 **۲۱ پلتفرم**       | سروش، ایتا، بله، روبیکا، آپارات، اینستاگرام، تلگرام و... |
| 📱 **QR کد**           | برای هر لینک و صفحه عمومی                                |
| 📊 **آمار بازدید**     | نمودار و شمارنده کلیک برای هر لینک                       |
| 🔍 **جستجو**           | جستجوی کاربران در صفحه اصلی                              |
| 🌙 **دارک/لایت مود**   | با قابلیت ذخیره تنظیمات                                  |
| 📱 **ریسپانسیو**       | تجربه عالی در تمام دستگاه‌ها                             |

## 🚀 اجرای سریع با Docker

```bash

git clone https://github.com/yourusername/linkhub.git
cd linkhub

# اجرا با docker-compose
docker-compose up -d
```
